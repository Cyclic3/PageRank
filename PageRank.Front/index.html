Explaination of the nsv (null-seperated values) format: Most websites contain commas and newlines, so a csv format could not work. I didn't need a complex database system, so I created my own format:
0x01 element 1 0x00 element 2 0x00 ... 0x00 element n-1 0x00 element n

acc


In this case, the website is the first value, the links are the second to penultimate value, and the final value is the text on that page. This is used to regenerated the PageRank for each page.

pr


This is a sorted array, with the first value being a URL, and the second being the text on that page. This is the main index used for searching.
